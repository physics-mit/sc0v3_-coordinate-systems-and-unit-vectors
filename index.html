Key Concepts (Based on Provided Text)

  • 2D Coordinate System: We use x and y axes to define positions and vectors on a plane ("grid"). This helps in performing vector operations systematically.
  • Unit Vectors: These are vectors with a magnitude (length) of exactly 1. They specify a direction.
    • i-hat (√Æ): A unit vector in the positive x-direction. Components: (1, 0).
    • j-hat (ƒµ): A unit vector in the positive y-direction. Components: (0, 1).
    • In Cartesian coordinates, √Æ and ƒµ are constant regardless of the point in space.
  • Unit Vector of an Arbitrary Vector (A): We can find a unit vector (uA) that points in the same direction as any non-zero vector A.
    • It's calculated by dividing vector A by its magnitude: uA = A / |A|.
    • This means if A = (ax, ay), then uA = (ax/|A|, ay/|A|).
    • The key is that the magnitude of uA will always be 1.
